Periodic Table Explorer :: Help
 
To open this page again search for help.
 

The toolbox has information on hundreds of physical constants, compounds, equations and the entire 1400+ pages of content, all of it fully searchable. Plus it can calculate the molecular mass of any compound.
 
Physical Constants
 
Click the search icon in the top left hand corner and select "Constants".
 
Searching for a constant is easy, just type the name or part of the description in the search box and click Search. mass, electron, pi and ratio are all valid search terms.
 
Compounds
 
Click the search icon in the top left hand corner and select "Compounds".
 
Search for a compound either by name, part of a name or constituent component. acid, ohol, Mg2, H2O, hexane and Li are all valid search terms.
 
Equations
 
Click the search icon in the top left hand corner and select "Equations".
 
Search for an equation by name. Clicking a link in the search results will open the selected equation.
 
Periodic Table Explorer Content
 
Click the search icon in the top left hand corner and select "Page Content".
 
It's possible to search the entire Periodic Table Explorer library of over 1400 pages with this option.
 
A search for german will return pages containing german, germany and germanium. Your search term must be two or more letters in length. Search results are ordered by keyword.
 
Click any link to open a page. Navigate by using page links or the < and > history buttons at the top right of the screen.
 
Molecluar mass calculator
 
Click the search icon in the top left hand corner and select "Molecluar calculator".
 
Enter any compound to calculate its molecular mass. H2O, CH3OH, CH3(CH)3, CCl4 and C6[CH3]6 are all valid.
 
The calculator doesn't currently support compounds in the form KMgCl3.6H2O. To calculate the mass of this compound enter the formula as KMgCl3(H2O)6.
